# ScrapNet Wiki

An open catalogue and companion app for community‑made AI gadgets, firmware and prompt packs.

---

## 1 · What is ScrapNet?

ScrapNet is a fully open‑source platform that lets anyone:

1. **Browse and share** hardware designs, firmware and AI prompt packs through a simple web catalogue.
2. **Pair gadgets** with a companion app (Android first) that routes data to any AI backend the user chooses—local or cloud.
3. **Contribute code, designs and ideas** via public repositories under permissive licences.

The website hosts the knowledge, the app is the bridge and devices are limited only by community imagination.

## 3 · Guiding Principles

* **Open by default** – source code, CAD files and docs public from day one.
* **BYOK (Bring Your Own Key)** – users decide which AI service or self‑hosted model they trust; ScrapNet stores no credentials.
* **Privacy & EU compliance** – GDPR/DSA aligned; minimal data collection; accounts only when strictly necessary.
* **Community ownership** – direction shaped by contributors, not commercial interests.

## 7 · Licensing Overview

| Asset type                       | Default licence  |
| -------------------------------- | ---------------- |
| Software (app, firmware, SDKs)   | **BSD‑3‑Clause** |
| Hardware designs (PCB, CAD, BOM) | **CERN‑OHL‑W**   |
| Documentation & media            | **CC‑BY‑SA 4.0** |

### Licence Summary

* **BSD‑3‑Clause** – Permissive: use, modify, merge or sell the code, including in closed‑source products, with attribution.
* **CERN‑OHL‑W** – Copyleft for hardware: if you share or manufacture modified designs, you must publish full design files under the same licence.
* **CC‑BY‑SA 4.0** – Share‑alike for docs and media: copy, remix and distribute—even commercially—with attribution and the same licence.

These licences ensure everyone can study, improve and redistribute ScrapNet while keeping improvements in the commons.
